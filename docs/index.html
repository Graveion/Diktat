diktat
say it. ship it.

Drive your Claude Code & Cursor coding sessions from your phone — from anywhere.

Install the daemon on your Mac

curl -fsSL https://graveion.github.io/Diktat/install.sh | bash
1

Install

Run the command above. It sets up Bun, the daemon, and the diktat command.

2

Set up

Run diktat setup to detect your CLIs and choose which projects to expose.

3

Pair

Run diktat pair — it shows a QR. In the app, tap Pair a machine and scan it.

4

Go

Run diktat start. Open the app, pick your Mac, and start a session from your phone.

Requires macOS, Bun, and Claude Code and/or the Cursor CLI installed and logged in.
